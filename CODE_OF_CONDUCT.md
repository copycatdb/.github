If someone violates this Code of Conduct:

### Reporting

If you experience or witness unacceptable behavior:

- **Email:** Report to the maintainer (contact info in the org profile)
- **GitHub:** Use the "Report" feature on any comment
- **DM:** Reach out to `@tinkeringforfun` directly

Reports are handled confidentially. We take this seriously, even if the rest of our project has cat puns in it.
